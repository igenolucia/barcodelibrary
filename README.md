# Barcode Library - NCBI/GenBank Data Extraction

Herramienta para la automatización de la descarga de secuencias de referencia de **GenBank** (NCBI) mediante la herramienta **BLAST**.

## ¿Para qué sirve?
Este script permite optimizar el flujo de trabajo para proyectos de **Barcoding**, facilitando:
* La obtención de secuencias de referencia.
* El filtrado y limpieza de datos biométricos y taxonómicos asociados.
* La preparación de bases de datos locales para asignación taxonómica.

## ¿QUé contiene?
* **Script de Consulta:** Código optimizado para interactuar con la API de NCBI.
* **Protocolo de Operación:** Documentación técnica sobre el flujo de trabajo y parámetros de búsqueda.
* **Licencia MIT:** Código abierto para uso en investigación científica.

## Contexto
Este módulo forma parte de un proyecto de generación de librerías de megabarcode, diseñado para mejorar la precisión en la identificación taxonómica mediante herramientas bioinformáticas.
